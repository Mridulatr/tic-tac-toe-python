# Tic Tac Toe - Python

This is a simple **Tic Tac Toe** game built with Python.

## Features
- Two-player game
- Text-based interface
- Easy to play and understand

## How to Run
1. Make sure you have Python installed.
2. Open the terminal/command prompt in the project folder.
3. Run the game:

```bash
python tictactoe.py
